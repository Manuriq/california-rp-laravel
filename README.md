## About Project

A web application created with Laravel for the San Andreas Multiplayer California RolePlay game server:

- [San Andreas MultiPlayer](https://www.sa-mp.com/).
- [Laravel](https://laravel.com/).

### Installation

```bash
    composer install
```

```bash
    npm install
```
